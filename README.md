## Installation
```
- git clone https://github.com/KianSantang777/STRIPE-AUTHX.git
- pip install aiohttp aiofiles aiohttp_socks pyfiglet faker requests httpx colorama psutil
- cd STRIPE-AUTHX
- python -m pip install -r requirements.txt
- python oki.py
```
